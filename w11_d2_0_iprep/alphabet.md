## Alphabet

Write a program that outputs the sorted alphabet given a sorted array of strings.

```js
var sortedArray = ["alan", "ben", "cameron", "jane"..."will"];
findAlphabet(sortedArray);
// => abcd...z
```

Your program should handle any 'alphabet': 

```js
var sortedArray = ["$*", "*#&", "*##"];
findAlphabet(sortedArray);
// => $*#&
```
